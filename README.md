# SQL-Internship-Task-11

#Indexing and Query Performance Optimization

- Created a sample database and table using MySQL Workbench.
- Executed SELECT queries without indexes to observe performance.
- Identified full table scans in query execution.
- Created indexes on frequently searched columns.
- Re-ran the same queries after index creation.
- Compared query performance before and after indexing.
- Used EXPLAIN to analyze query execution plans.
- Observed index usage and reduced rows scanned.
- Learned clustered and non-clustered index concepts.
- Identified scenarios where indexes should be avoided.
- Documented basic indexing best practices.
